# Portfolio

React + Vite + MUI 기반 미니멀 포트폴리오 사이트.

- 로딩 화면 → 메인 페이지(섹션1: 가상 프로필 / 섹션2: 작업물 목차) 구성
- 디자인 시스템: `color_pallette/DIETRO_ColorSystem.md`(컬러), `DesignSystem/BOOK_DesignSystem.md`(타이포/간격/라운드) 참고

## 개발

```bash
npm install
npm run dev
```

## 배포

GitHub Actions 워크플로우(`.github/workflows/deploy.yml`)를 통해 GitHub Pages로 자동 배포된다.
